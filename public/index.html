Viva la Libris

Book sharing database api

POST /user

  • firstName (string, required)
  • lastName (string, required)
  • email (string, required)
  • password (string, required, must be 8 characters)

Successful requests will recieve the new user data with the password removed, plus user _id.

POST /auth/login

  • email
  • password

Successful requests will be issued with a jwt valid for one day.
